# WFChat

## Install

```bash
npm install
```

## Run Frontend

```bash
npm run dev:web
```

Frontend: `http://localhost:5173`

## Run Backend

```bash
npm run dev:api
```

Backend API: `http://localhost:8080`

## Build And Check

```bash
npm run build
```

## Docker

```bash
docker compose up --build
```

Docker frontend: `http://localhost:5173`

Docker backend API: `http://localhost:8080`
