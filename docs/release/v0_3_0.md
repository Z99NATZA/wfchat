# v0.3.0 - Aiko Cafe

Release date: 2026-07-19

This release introduces Aiko Cafe, a guest-first realtime co-op space where up
to eight players can prepare tea together. Login is optional.

## Highlights

- Join an available room instantly, create a room, or join friends with a
  six-character invite code.
- Move and see other players in real time with desktop and mobile controls,
  preset emotes, player names, and equipped cosmetics.
- Collect tea leaves and deliver them to Aiko in replayable shared rounds.
- Earn persistent Cafe Stars and unlock the Sakura Pin, Mint Scarf, and Tea Hat.
- Start with a generated name or choose a temporary name for the current tab.
- Learn the activity through first-visit guidance, nearby interaction prompts,
  map markers, and an always-available help action.

## Progress And Reliability

- Guest Cafe progress transfers to the account when the player signs in.
- Round rewards are recorded once per eligible player, even after reconnects.
- Movement and activity state are server-authoritative.
- Offline input stops immediately and the room reconnects automatically with a
  fresh server snapshot before controls resume.
- The camera keeps the game readable on smaller viewports by following the
  player instead of shrinking the entire room.

## Upgrade Notes

- No breaking configuration changes or new environment variables are required.
- Existing chat, memory, and account data are retained.
- The API applies the three Cafe database migrations automatically on startup.
- Docker users can upgrade with:

  ```bash
  docker compose up -d --build
  ```

- No additional service, port, or volume is required.

## Known Limitations

- Active rooms are held in one API process and end when that process restarts;
  multi-instance room sharing is not supported.
- The current release has one room map, one tea activity, and three cosmetics.
- Public free-text chat, moderation tools, AI-generated room dialogue, regional
  matchmaking, and spectator mode are not included.
