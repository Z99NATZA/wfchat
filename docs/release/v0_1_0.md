# v0.1.0 - Initial Release

First baseline release of **WaifuChat**, a full-stack companion chat app with a
React frontend, Rust API, and PostgreSQL database.

## Highlights

- Chat interface with session-based conversations
- Streaming assistant responses
- Markdown rendering for assistant messages
- Quick prompts from backend chat UI configuration
- Optional Google sign-in and profile editing
- App settings for assistant voice playback preferences
- Docker Compose setup for local and LAN testing

## Voice Features

- Assistant text-to-speech playback
- Push-to-talk speech-to-text input from the chat composer
- Compact inline recording controls inside the input row
- Backend-owned voice and transcription provider configuration
- OpenAI speech-to-text support
- OpenAI and VOICEVOX text-to-speech support
- VOICEVOX Japanese speech text policy for natural spoken output
- App-level voice credits for providers such as VOICEVOX

## Avatar Features

- PNGTuber workspace with Aiko avatar assets
- Basic assistant speech motion bridge
- Avatar can switch to semantic `thinking`, `talking`, and `idle` states during
  assistant voice playback

## Developer Notes

- Frontend: React + TypeScript + Vite
- Backend: Rust + Axum
- Database: PostgreSQL
- Local development and deployment are supported through Docker Compose

## Known Limitations

- Realtime voice conversation is not included
- Avatar lip sync and viseme animation are not included
- Provider, model, and API key controls are intentionally not exposed in the
  normal chat UI
- Some features are still intended for local or experimental use

## Upgrade Notes

This is the first tagged release, so there are no migration steps from a
previous version.
