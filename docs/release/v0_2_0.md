# v0.2.0 - Automatic Memory And Follow-Ups

Release date: 2026-07-16

WaifuChat can now remember useful details from previous conversations and bring
them back naturally when they become relevant. Aiko can also check in on a
meaningful plan or goal when the user opens a New Chat.

## Highlights

- Added selective automatic memory across conversations.
- Added relevant memory recall for both Thai and English topics.
- Added personalized New Chat follow-ups for meaningful unresolved plans and
  goals.
- Added time-aware memory for plans described with relative dates such as
  “tomorrow”.
- Added a Settings action to clear learned context while keeping chat history.

## Personalized Follow-Ups

- An empty New Chat may open with one personalized question based on something
  the user previously planned or wanted to achieve.
- Follow-ups are available in English and Thai and use the language currently
  selected in the app.
- Only current, meaningful, and sufficiently reliable plans or goals are used.
  Completed, cancelled, expired, unsafe, or low-confidence details are skipped.
- There is no generic fallback question when nothing suitable is available.
- Merely displaying a follow-up does not create a chat. A chat is created only
  when the user replies, and the opening question is then saved with the
  conversation.
- The same remembered item is not shown again, and follow-ups are limited to
  one per user and character within a rolling 24-hour period.
- The limit is shared across browsers for signed-in users and remains isolated
  per session for guests.
- If a follow-up cannot be loaded, New Chat continues to work normally.

## Automatic Memory

- Aiko selectively learns durable preferences, goals, plans, constraints, and
  experiences from successful conversations.
- Relevant memories can be reused across chats for the same user and character.
- Thai and English topic wording can match the same remembered subject, such as
  `เพลง` and `music`.
- Repeated information can strengthen a memory, while an explicit correction
  can replace an older value.
- Time-sensitive plans follow the user's timezone and stop being eligible after
  they expire.
- Memory capture runs in the background and does not delay a successful chat
  response.

## Privacy And Control

- Learned context is isolated by user and character.
- Secrets, credentials, financial identifiers, unsupported guesses, and prompt
  injection content are excluded.
- Aiko treats remembered context as optional guidance and prefers the latest
  user message when information conflicts.
- Clearing learned context removes remembered details and follow-up state while
  retaining existing chat history.

## Upgrade Notes

- Existing chat history is retained.
- Automatic memories are learned from conversations processed after upgrading;
  retired manual facts and summaries are not migrated.
- Database updates are applied automatically when the API starts.
- No vector database, embedding service, or additional infrastructure is
  required.

## Known Limitations

- Memory capture is selective and asynchronous, so not every detail is saved or
  available immediately.
- Thai-English matching covers a bounded set of supported topics rather than
  unrestricted semantic matching.
- Learned context is not shared between different characters.
- Individual memories cannot currently be viewed, edited, or deleted
  separately.
