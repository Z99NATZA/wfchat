Stack
Frontend: ReactJS + TypeScript
Backend: Rust + Axum
Database: PostgreSQL

Install
git clone https://github.com/Z99NATZA/wfchat.git
cd wfchat
npm run init

npm run init creates or updates local env files

Docker
Edit .env
VITE_GOOGLE_CLIENT_ID=

Edit apps/api/.env
OPENAI_API_KEY=
GOOGLE_CLIENT_ID=

Run
docker compose up -d --build

URLs default
Web: http://localhost:5173
API: http://localhost:8080

Stop services
docker compose down

More Docker details: `docs/docker.md`
